# WineWise Quizzes

**AI-powered wine education and quiz platform built with Next.js, TypeScript, Google Genkit, and Gemini.**

WineWise Quizzes helps users learn wine theory through interactive quiz modes, AI-generated questions, offline question banks, and answer explanations. The project is positioned as a portfolio-ready AI web application that combines practical product thinking with modern frontend engineering.

## Project Status

This is an active portfolio project. The current version demonstrates the core product experience: quiz generation, multiple play modes, local/offline fallback, and AI-assisted learning flows.

---

## Features

- **AI Mode** — generates unique wine quiz questions with Google Gemini.
- **Local Mode** — works without an API key using a built-in offline question bank.
- **Single Player Mode** — choose category, difficulty, number of questions, and timer.
- **Multiplayer Mode** — two-player turn-based quiz experience on one device.
- **Arcade Mode** — fast question flow for quick practice sessions.
- **Learning Mode** — explains why an answer is correct or wrong.
- **Custom Quiz Generator** — paste your own text and generate a quiz from it.
